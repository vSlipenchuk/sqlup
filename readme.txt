Логика простая, есть таблица

create table txt(w1 varchar(80), w2 varchar(80),w3 varchar(80), w4 varchar(80) );

мы считываем то, что пришло в тексте и грузим в эту таблицу.
Потом - можно вытащить то, что считаем интересным - через селект и распечатать его так, как нужно...


Основная идея - загружать консольный вывод во временные таблицы
для переформатирования средствами sql.

Может позже получится более мощный загрузчик.
